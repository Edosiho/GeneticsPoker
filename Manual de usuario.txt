Para hacer uso de la mejor mano de poker usando el método genético (poker.rb) es necesario tener instalado el  framework Shoes que se utiliza para crea GUIs utilizando el lenguaje Ruby. Después de tener instalado el frame work se puede correr el programa usando el comando: shoes poker.rb; otro método es abrir la aplicación de Shoes y seleccionar poker.rb en "Open an App". esto nos mostrara la GUI en donde se puede escoger el número de generaciones deseadas además si se quieren mostrar todas las poblaciones generadas o solamente la última. Finalmente, al presionar el botón start, se mostrarán las generaciones dadas, las poblaciones en caso de que la opción haya sido seleccionada, las estadísticas de cada generación y al final se mostrará la mano más alta. Cabe mencionar que el framework tiene problemas en Windows 7.
